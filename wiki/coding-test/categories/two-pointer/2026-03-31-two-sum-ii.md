## 문제
- **링크**: https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/
- **설명**: 오름차순 정렬된 배열에서 두 수의 합이 target인 두 수의 1-based 인덱스 반환 (O(1) space)
- **적정 풀이시간**: 20~35분

## 풀이
### 접근 방식
양끝 포인터 활용. left=0, right=끝에서 시작하여 합이 크면 right--, 작으면 left++.

### 코드
```typescript
function answer(numbers: number[], target: number): number[] {
    let leftPointer = 0;
    let rightPointer = numbers.length - 1;
    while (leftPointer !== rightPointer) {
        if (numbers[leftPointer] + numbers[rightPointer] === target) {
            return [leftPointer + 1, rightPointer + 1];
        }
        if (numbers[leftPointer] + numbers[rightPointer] > target) {
            rightPointer--;
        } else {
            leftPointer++;
        }
    }
    return [];
}
```

### 정확성
- 실제 제출: 통과
- 잠재 취약점: 특이사항 없음

### 시간/공간 복잡도
- 시간: O(n) / 공간: O(1)
- 이미 최적해

### 개선점
1. `leftPointer !== rightPointer` → `left < right`로 변경 (관용적 + 안전)
2. TypeScript 타입 어노테이션 추가
3. sum 변수로 중복 계산 제거, 변수명 간결화

### 개선된 코드
```typescript
function twoSum(numbers: number[], target: number): number[] {
  let left = 0;
  let right = numbers.length - 1;

  while (left < right) {
    const sum = numbers[left] + numbers[right];
    if (sum === target) return [left + 1, right + 1];
    if (sum > target) right--;
    else left++;
  }

  return [];
}
```

### 핵심 takeaway
정렬된 배열에서 두 수의 합 → 양끝 포인터로 O(n) 해결. 1-based vs 0-based 인덱스 항상 확인.
