### 개선된 코드
```typescript
function subsets(nums: number[]): number[][] {
  const result: number[][] = [];

  function backtrack(index: number, current: number[]) {
    result.push([...current]);  // 현재 부분집합 저장

    for (let i = index; i < nums.length; i++) {
      current.push(nums[i]);    // 선택
      backtrack(i + 1, current);
      current.pop();            // 선택 취소 (backtrack)
    }
  }

  backtrack(0, []);
  return result;
}
// 시간: O(n × 2ⁿ), 공간: O(n)
```

### 핵심 takeaway
result.push([...current])를 반복문 앞에서 호출해 빈 집합부터 누적 저장; current.pop()으로 선택 취소가 backtracking의 핵심

### 흔한 실수: `i+1` vs `index+1`

```typescript
// 틀린 코드 — index 고정
backtrack(index + 1, current);
// i=2일 때도 항상 index+1로 돌아가 이미 쓴 원소 재방문
// 결과: [3,3,3] 같은 중복 부분집합 생성 (총 16개)

// 올바른 코드 — i 기준

// "현재 선택한 원소 다음부터" 탐색 보장 (총 8개)
```
